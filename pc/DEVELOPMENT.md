# 电脑系统下载中心 - 开发文档

## 项目概述
这是 kaysarsoft 项目的电脑系统下载中心页面，提供现代化的软件资源下载体验。

## 文件结构
```
pc/
├── index.html              # 主页面文件
├── style_optimized.css     # 优化后的样式文件
├── style.css              # 基础样式文件（备用）
├── style_new.css          # 新样式文件（备用）
├── idm激活.cmd            # IDM激活脚本
├── screenshot.exe         # 截图工具
└── DEVELOPMENT.md         # 开发文档（本文件）
```

## 样式系统架构

### CSS变量系统
项目使用现代CSS变量系统，包含以下类别：

**颜色变量**
- `--primary`: 主色调 (#667eea)
- `--secondary`: 辅助色 (#764ba2)  
- `--accent`: 强调色 (#f093fb)
- `--success`: 成功色 (#4ecdc4)
- `--text-dark`: 深色文本 (#2d3436)
- `--text-light`: 浅色文本 (#ffffff)

**背景和视觉效果**
- `--bg-gradient`: 背景渐变
- `--glass-bg`: 玻璃态背景
- `--glass-border`: 玻璃态边框
- `--group-bg`: 分组背景

**阴影系统**
- `--shadow-light`: 轻微阴影
- `--shadow-medium`: 中等阴影
- `--shadow-heavy`: 重度阴影
- `--shadow-colored`: 彩色阴影

**动画和交互**
- `--transition`: 标准过渡动画
- `--transition-fast`: 快速过渡动画

## 开发指南

### 本地开发
```bash
# 启动本地服务器
python -m http.server 8080
# 访问 http://localhost:8080/kaysarsoft/pc/
```

### 代码规范
1. 使用CSS变量定义颜色和尺寸
2. 采用BEM命名规范
3. 保持代码结构清晰和注释完整
4. 优先考虑性能和可维护性

### 维护建议
1. 定期更新依赖资源
2. 监控页面性能指标
3. 保持CSS代码的模块化
4. 持续优化用户体验
